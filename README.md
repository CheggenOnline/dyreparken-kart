# dyreparken-kart

Interaktivt familiekart over Dyreparken i Kristiansand.

Live: https://cheggenonline.github.io/dyreparken-kart/

- Sanntids GPS-posisjon og gangtider i 4-åringstempo
- Raskeste vei langs stiene (OSRM foot routing)
- Kan legges til på Hjem-skjerm som app (PWA, fullskjerm med eget ikon)
